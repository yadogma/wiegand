# wiegand
simple wiegand 26 converter on PHP

Convert Wiegand26 [DEC] to Weigand

**for example request URL:**

wiegand26.php?q=2817931

**It will output:**

42/65419

wiegand26.php?e=2817931

**It will output:**

2817931 = '1010101111111110001011'
65419 = '1111111110001011'
171 = '10101011'
171/65419
2817931 = 2817931
END
